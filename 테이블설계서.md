# 테이블 설계서

## MEMBER 회원 테이블

> 데이터 컬럼

| 컬럼명           |  데이터타입   | 설명                             |
| :--------------- | :-----------: | :------------------------------- |
| MEM_NO           |    NUMBER     | 회원번호, 시퀀스 사용            |
| MEM_ID           | VARCHAR2(30)  | 사용자 식별 값(API에서 제공 UID) |
| MEM_EMAIL        | VARCHAR2(30)  | 회원이메일, 필수항목             |
| REFRESH_TOKEN    | VARCHAR2(30)  | 리프레쉬 토큰                    |
| PLATFORM         | VARCHAR2(30)  | DEFAULT 'NORMAL'                 |
| MEM_ADDRESS      | VARCHAR2(100) | 필수항목                         |
| MEM_PHONE        | VARCHAR2(20)  | 필수항목                         |
| NICKNAME         | VARCHAR2(30)  | 기본값 이메일 + 난수 자바로 처리 |
| REGISTERED_DATE  |     DATE      | DEFAULT SYSDATE, 가입날짜        |
| UPDATED_DATE     |     DATE      | 정보수정 날짜                    |
| STATUS           |    CHAR(1)    | DEFAULT 'Y'                      |
| DEACTIVATED_DATE |     DATE      | 비활성화 날짜                    |
